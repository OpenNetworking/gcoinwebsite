GCOIN 區塊鏈協議致力於提供一套開放的數位金融基礎建設，使用 GCOIN 可打造出支援兼顧安全性、效率性、低維護成本的交易平台。區塊鏈上的數位化資產能夠與真實世界中的各種資產建立不可竄改的連結，包括點數、票據、證券、權證、甚至房地產等線下資產。

GCOIN 將區塊鏈針對金融應用與電子商務所需的交易速度與可擴展性加以改進並優化，可真正實現數位金融的虛實整合。

分散式數位帳本 --- 區塊鏈

「區塊鏈 (Blockchain)」從 2015 年起成為全球金融科技創新的焦點，而 GCOIN 本身就是一套開源的區塊鏈底層協議。區塊鏈將所發生的交易紀錄到區塊 (Block) 中，並藉由網絡參與者共同驗證的方式來確保交易的完整性，採用這項技術賦予金融機構去中介化、高安全性、結算速度快、成本較低、可供監察驗證和資料最終性等優點。GCOIN 改良原本稱為「挖礦 (Mining)」的分散式共識演算法 (Decentralized Consensus Algorithm)，大幅提升區塊鏈的交易速度與可擴展性。

支援多元資產並存

GCOIN 區塊鏈上可以存在多種數位資產，這是過去的比特幣區塊鏈缺乏之功能。GCOIN 網絡中同時承載多種數位資產，並且支援極大之發行總量上限，而非比特幣之兩千一百萬貨幣總量限制。GCOIN 藉由區塊鏈上的聯盟成員賦予資產發行權限，每個資產發行憑證可發行一種資產，而發行者也可將對應的資產訊息紀錄在區塊鏈上，作為不可竄改的連結。例如，NASDAQ 使用 LINQ 紀錄私募股權市場的股票，就是應用區塊鏈發行數位資產的功能。

支援高頻交易

相較於過去比特幣目前只能負荷平均每秒 7 個交易的每秒交易數 (TPS) 以致無法承載高頻交易，GCOIN 區塊鏈針對此問題將交易速度做優化，交易高峰時平均每 10-15 秒將交易記錄到區塊。GCOIN 為滿足金融應用所作的特殊設計可大幅提升可承載之交易頻率，且大幅 縮短交易確認時間。

改良的演算法

為了避免匿名參與者的惡意攻擊，並解決比特幣區塊鏈之 51% 攻擊問題，GCOIN 區塊鏈針對認許制或實名制網絡將既有的分散式共識演算法進行改良，大幅降低分散式系統發生獨佔驗證的可能性，更適合做為數位金融基礎建設。更多細節請參考 Github 上的開放原始碼。

多中心階層架構

GCOIN 區塊鏈設計不同角色的參與者維持網絡的功能完整性。網絡中的聯盟成員(Alliance members) 負責驗證交易之正確性，生成區塊供網絡獲取交易歷史資訊。發行者 (Issuers) 自聯盟成員獲得發行許可，負責產生各種數位資產。錢包 (Wallets) 可用來儲存數位資產，使用戶直接存取，進行交易、獲取資產資訊等，而不需透過中介商。
