Gcoin is an open protocol and platform that makes financial market more transparent. Gcoin network can be inspected and verified by all participating users. Traditional financial instruments can be brought onto Gcoin platform, doing away the need for intermediary parties and the associated inconveniences and dangers. Gcoin has improved the Bitcoin blockchain technology and transformed it into a transaction platform with greater capacity for expansion and higher transaction frequency.

Our goal is to use blockchain technology to construct a comprehensive state-of-the-art peer-to-peer financial transaction platform.

A secure distributed platform

Similar to Bitcoin, Gcoin employs distributed ledger system called “blockchain”. All verified transactions are placed on blockchain. The transactions are verified by all network participants. A process called “mining” is used to verify the transaction, maintain transaction chronological orders, and achieve consensus among decentralized network participants.

Multiple electronic currencies

Gcoin is a multi-currency network. Bitcoin is a single-currency network. Gcoin natively supports multiple currencies. Gcoin currency issuers have to obtain a coin-minting license to issue the currency. A user may need to register with an issuer to use that issuer’s currency.

Supports high-frequency transactions

One of the limitations of Bitcoin is its fairly low transactions per second (TPS). Currently, Bitcoin network can only handle an average transaction volume of 7 TPS. Gcoin network addressed this challenge by having shorter block time. On average, at peak transaction times, Gcoin network generates a transaction block every 15 seconds. The Gcoin network has been specifically designed to satisfy financial applications. It has substantially increased its transaction frequency capacity and substantially shortened transaction confirmation time.

A more secure algorithm

The Gcoin network improves upon Bitcoin’s proof of work algorithm with the alliance-based CPOW (Consortium proof of work) algorithm. Combind with dynamic mining difficulty improvement, Gcoin is able to avoid the famous “51% attack” vulnerability in Bitcoin network.

Multi-centered decentralized structure

Participants taking on different roles to maintain the platform’s functional integrity. Multiple alliance members take on the responsibility of generating blocks. These blocks provide historical transaction information. Alliance member nodes also inspect and verify network transactions.

Issuers have coin-minting certificates that can issue a Gcoin currency. Users (Gcoin Wallet) need to register with individual issuer to use that issuer’s currency.

Users can obtain currency from an issuer or another user. (see Gcoin White paper)
